# 이미지 업로드 폴더

UI에서 쓰거나, 제가(클로드가) 확인해야 하는 **이미지를 여기에 올려 두는 곳**입니다.
빌드하면 이 폴더의 이미지가 앱 번들(`assets/`)로 들어갑니다.

## 폴더 구성

- **`brand/`** — 로고 등 브랜드 이미지
  - `logo.png`(또는 `logo.svg`, `logo.jpg`)를 넣으면 **앱 로고로 자동 적용**됩니다.
    파일이 없으면 기본 알파(α) 로고가 표시됩니다.
- **`ui/`** — 그 밖에 UI에 넣을 그림 / 제가 확인했으면 하는 참고 이미지
  - 자유롭게 올리시고, 각 이미지를 **어디에 어떻게 쓰고 싶은지** 말씀해 주시면
    해당 위치에 넣어 드립니다.

## 쓰는 법 (요약)

1. 위 폴더에 이미지 파일을 커밋(업로드)합니다.
2. 로고면 `brand/logo.*` 이름으로 두면 자동 반영됩니다.
   그 외 이미지는 용도를 알려주세요.
3. 배포 사이트(GitHub Pages)에 반영되려면 **다시 빌드**가 필요합니다.
   올리신 뒤 "반영해줘"라고 하시면 빌드해서 PR로 올려 드립니다.

> 지원 형식: PNG · SVG · JPG · JPEG · WEBP
