# ui — UI 이미지 / 참고 이미지 폴더

로고 외에 **UI에 넣고 싶은 그림**이나, 제가(클로드가) **확인했으면 하는 이미지**를 여기에 올려 주세요.

## 예시

- 배너/일러스트/아이콘 등 화면에 넣고 싶은 이미지
- 원하는 디자인 시안·스크린샷·손그림 (제가 보고 반영)
- 시험지 예시 이미지 등 참고 자료

## 방법

1. 이미지를 이 폴더에 올립니다(알아보기 쉬운 이름 권장, 예: `home-banner.png`).
2. **어디에, 어떻게** 쓰고 싶은지 한 줄로 알려 주세요.
   (예: "home-banner.png를 홈 상단 배경으로 넣어줘")
3. 제가 해당 위치에 반영하고, 필요하면 빌드해서 PR로 올립니다.

> 지원 형식: PNG · SVG · JPG · JPEG · WEBP
