# brand — 로고 · 직인 폴더

여기에 **로고 파일 하나**를 넣으면 앱 로고로 자동 적용됩니다.
**직인(도장) 이미지**(`seal.png` 등)를 넣으면 리포트 PDF 하단 직인으로 쓰이며, 없으면 자동 생성 도장이 표시됩니다.

## 방법

이 폴더에 아래 이름 중 하나로 로고를 올려 주세요:

- `logo.png` (권장 — 배경 투명 PNG)
- `logo.svg`
- `logo.jpg` / `logo.jpeg` / `logo.webp`

파일을 넣으면 홈 화면과 각 화면 상단의 로고가 **업로드한 로고로 교체**됩니다.
아무 파일도 없으면 기본 알파(α) 로고가 나옵니다.

## 참고

- 정사각형에 가까운 이미지가 가장 잘 맞습니다(로고는 정사각 영역에 맞춰 표시).
- 배포 사이트 반영은 재빌드가 필요합니다 — 올리신 뒤 알려주시면 빌드해서 PR로 올립니다.
- 두 개 이상 올리면 그중 하나만 사용됩니다. 하나만 두는 것을 권장합니다.
